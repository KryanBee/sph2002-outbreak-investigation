# Outbreak Investigation Report

This repository contains an R Markdown notebook that generates a report for an epidemiological outbreak investigation project in SPH2002.

## To run the notebook:

1. Install **R** and **RStudio**  
2. Open the `.Rmd` file in RStudio  
3. *(Optional)* To generate the report with code blocks shown, set the following in the YAML header:

```yaml
params:
  show_code: TRUE
```

4. Click **Knit** to generate the report in `src/`. A copy of the report without code blocks has already been pre-generated.
